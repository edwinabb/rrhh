1. Descripción General

Objetivo MVP

Implementar un módulo completo de gestión de legajos digitales por empleado que cumple con la normativa peruana de protección de datos personales (Ley 29733 PDPA). El sistema permite centralizar documentación laboral, mantener histórico de cambios, asegurar trazabilidad mediante audit logs, y controlar acceso mediante RBAC.

Funcionalidades Priorizadas

Legajo Digital Organizado
Carpeta virtual por empleado con estructura jerárquica de tipos de documentos (Contrato, CV, ID, Certificados, etc.)
Versionamiento Automático
Historial completo de cambios: quién subió, cuándo, qué versión anterior. Auditable 100%.
Búsqueda Avanzada
Filtrar por empleado, tipo de documento, rango de fechas, uploader. Indexación en Prisma.
Descarga ZIP del Legajo
Generación de archivo ZIP con estructura del legajo completo (solo documentos con acceso permitido).
Almacenamiento Escalable
MinIO S3-compatible. Metadatos en PostgreSQL, blobs en object storage. Separación clara.
RBAC Granular
Admin (full), RRHH (read+upload), Manager (read own reports). Integrado con RLS.
Cumplimiento PDPA Perú: Todos los documentos son datos personales. El sistema registra consentimiento, encriptación en tránsito (TLS), auditoría de acceso (audit log), y facilita cumplimiento de derecho al olvido mediante soft-delete + hard-delete asincrónica.

No Incluir en Fase 3

  • Firmas digitales avanzadas (RSA, certificados PKI)
  • Workflow de aprobación multi-step
  • ESS Sunat (Expediente Simplificado Sunat) - Fase 4
  • OCR y extracción de datos
  • Integración con suscriptor de TER (Terceros)

2.2 RLS (Row Level Security) Strategy

Implementar en PostgreSQL mediante políticas:

-- PostgreSQL RLS Policies

-- Policy: Multi-tenant isolation
CREATE POLICY tenant_isolation ON "Document"
  AS RESTRICTIVE
  USING (
    "tenantId" = current_setting('app.current_tenant_id')
  );

-- Policy: RBAC-based access
CREATE POLICY access_by_role ON "Document"
  AS PERMISSIVE
  USING (
    -- Admin: acceso total
    current_setting('app.user_role') = 'ADMIN'
    OR
    -- RRHH: ver todos, pero no empleados de otros tenants
    (current_setting('app.user_role') = 'RRHH' AND "tenantId" = current_setting('app.current_tenant_id'))
    OR
    -- Manager: solo sus reportes (implementar en app layer con join a organizational hierarchy)
    (current_setting('app.user_role') = 'MANAGER' AND "employeeId" IN (
      SELECT id FROM "Employee" WHERE "managerId" = current_setting('app.user_id')
    ))
    OR
    -- Employee: solo su propio legajo
    (current_setting('app.user_role') = 'EMPLOYEE' AND "employeeId" = current_setting('app.user_id'))
  );

-- Enable RLS
ALTER TABLE "Document" ENABLE ROW LEVEL SECURITY;
Nota sobre RLS en Prisma

Prisma no controla directamente RLS. Usaremos:
1. RLS a nivel PostgreSQL (políticas SQL)
2. Context de tenant en cada query: @db.Raw() con SET app.current_tenant_id
3. Validación en app layer (NestJS) antes de queries

3.1 MinIOIntegration Service

Gestionar conexión con MinIO y operaciones S3-compatible.

// src/modules/documents/services/minio.integration.ts

import { Injectable, Logger, BadRequestException } from '@nestjs/common';
import * as Minio from 'minio';
import { ConfigService } from '@nestjs/config';

@Injectable()
export class MinIOIntegrationService {
  private minioClient: Minio.Client;
  private logger = new Logger(MinIOIntegrationService.name);

  constructor(private configService: ConfigService) {
    this.minioClient = new Minio.Client({
      endPoint: this.configService.get('MINIO_ENDPOINT'),
      port: this.configService.get('MINIO_PORT'),
      useSSL: this.configService.get('MINIO_USE_SSL') === 'true',
      accessKey: this.configService.get('MINIO_ACCESS_KEY'),
      secretKey: this.configService.get('MINIO_SECRET_KEY'),
      region: this.configService.get('MINIO_REGION', 'us-east-1'),
    });

    this.initializeBuckets();
  }

  /**
   * Inicializar buckets tenants si no existen
   */
  private async initializeBuckets(): Promise {
    const bucketName = this.configService.get('MINIO_BUCKET');
    try {
      const exists = await this.minioClient.bucketExists(bucketName);
      if (!exists) {
        await this.minioClient.makeBucket(bucketName, 'us-east-1');
        this.logger.log(`Bucket ${bucketName} created`);
      }
    } catch (error) {
      this.logger.error(`Failed to initialize bucket: ${error.message}`);
    }
  }

  /**
   * Subir archivo a MinIO
   * @param fileBuffer Buffer del archivo
   * @param minioKey Ruta en MinIO
   * @param mimeType Content-Type
   * @returns URL de acceso y metadatos
   */
  async uploadFile(
    fileBuffer: Buffer,
    minioKey: string,
    mimeType: string,
  ): Promise<{ url: string; size: number; etag: string }> {
    try {
      const bucketName = this.configService.get('MINIO_BUCKET');

      // Validaciones
      if (fileBuffer.length > 100 * 1024 * 1024) { // 100MB max
        throw new BadRequestException('File size exceeds 100MB limit');
      }

      const etag = await this.minioClient.putObject(
        bucketName,
        minioKey,
        fileBuffer,
        fileBuffer.length,
        {
          'Content-Type': mimeType,
          // Encriptación en tránsito, metadatos
          'x-amz-meta-uploaded-at': new Date().toISOString(),
        },
      );

      const url = await this.minioClient.presignedGetObject(
        bucketName,
        minioKey,
        7 * 24 * 60 * 60, // 7 días de expiración
      );

      return {
        url,
        size: fileBuffer.length,
        etag,
      };
    } catch (error) {
      this.logger.error(`Upload failed: ${error.message}`);
      throw error;
    }
  }

  /**
   * Descargar archivo desde MinIO
   */
  async downloadFile(minioKey: string): Promise {
    try {
      const bucketName = this.configService.get('MINIO_BUCKET');
      const chunks: Buffer[] = [];

      const dataStream = await this.minioClient.getObject(bucketName, minioKey);

      return new Promise((resolve, reject) => {
        dataStream.on('data', (chunk) => chunks.push(chunk));
        dataStream.on('end', () => resolve(Buffer.concat(chunks)));
        dataStream.on('error', reject);
      });
    } catch (error) {
      this.logger.error(`Download failed: ${error.message}`);
      throw error;
    }
  }

  /**
   * Eliminar archivo (hard delete)
   */
  async deleteFile(minioKey: string): Promise {
    try {
      const bucketName = this.configService.get('MINIO_BUCKET');
      await this.minioClient.removeObject(bucketName, minioKey);
    } catch (error) {
      this.logger.error(`Delete failed: ${error.message}`);
      throw error;
    }
  }

  /**
   * Generar URL de descarga con expiración
   */
  async getPresignedUrl(minioKey: string, expirySeconds = 3600): Promise {
    try {
      const bucketName = this.configService.get('MINIO_BUCKET');
      return await this.minioClient.presignedGetObject(bucketName, minioKey, expirySeconds);
    } catch (error) {
      this.logger.error(`Presigned URL generation failed: ${error.message}`);
      throw error;
    }
  }
}

  /**
   * Subir documento (crea o nueva versión)
   */
  async uploadDocument(dto: UploadDocumentDto): Promise {
    const { file, tenantId, employeeId, documentType, title, uploadedBy } = dto;

    // Validar tenant context
    await this.validateTenantContext(tenantId);

    // Validar empleado existe en tenant
    const employee = await this.prisma.employee.findUnique({
      where: { id: employeeId },
    });

    if (!employee || employee.tenantId !== tenantId) {
      throw new NotFoundException('Employee not found');
    }

    // Generar ruta MinIO
    const documentId = crypto.randomUUID();
    const versionNumber = 1;
    const minioPath = this.generateMinIOPath(tenantId, employeeId, documentId, versionNumber);
    const minioKey = `${minioPath}/${file.originalname}`;

    // Subir a MinIO
    const { size, etag } = await this.minioService.uploadFile(
      file.buffer,
      minioKey,
      file.mimetype,
    );

    // Calcular checksum
    const checksum = crypto.createHash('sha256').update(file.buffer).digest('hex');

    // Crear documento en BD
    const document = await this.prisma.document.create({
      data: {
        id: documentId,
        tenantId,
        employeeId,
        type: documentType,
        title,
        description: dto.description,
        minioPath,
        minioKey,
        fileSize: BigInt(size),
        mimeType: file.mimetype,
        uploadedBy,
        requiresConsent: dto.requiresConsent ?? true,
        consentDate: dto.consentDate,
        consentSource: 'system_upload',
        currentVersionId: undefined, // Se actualiza después
      },
    });

    // Crear versión
    const version = await this.prisma.documentVersion.create({
      data: {
        documentId: document.id,
        versionNumber,
        minioPath,
        minioKey,
        fileSize: BigInt(size),
        mimeType: file.mimetype,
        uploadedBy,
        uploadReason: dto.uploadReason,
        isCurrentVersion: true,
        checksum,
      },
    });

    // Actualizar documento con versión actual
    const updatedDoc = await this.prisma.document.update({
      where: { id: document.id },
      data: { currentVersionId: version.id },
    });

    // Audit log
    await this.auditService.logAction({
      tenantId,
      documentId: document.id,
      action: 'UPLOAD',
      performedBy: uploadedBy,
      changeDetails: { fileName: file.originalname, size, type: documentType },
    });

    return updatedDoc;
  }

  /**
   * Descargar archivo específico

  async downloadDocument(
    documentId: string,
    tenantId: string,
    requestedBy: string,

      tenantId,
      documentId,
      action: 'DOWNLOAD',
      performedBy: requestedBy,
    });

    return fileBuffer;
  }

  /**
   * Descargar legajo completo como ZIP
   */
  async downloadLegajo(dto: DownloadLegajoDto): Promise {
    const { tenantId, employeeId, requestedBy } = dto;

    // Validar acceso
    await this.validateEmployeeLegajoAccess(tenantId, employeeId, requestedBy);

    // Obtener todos los documentos activos del legajo
    const documents = await this.prisma.document.findMany({
      where: {
        tenantId,
        employeeId,
        status: DocumentStatus.ACTIVE,
        deletedAt: null,
      },
      include: {
        currentVersion: true,
      },
      orderBy: [{ type: 'asc' }, { uploadedAt: 'desc' }],
    });

    // Crear ZIP
    const zip = new JSZip();
    const now = new Date().toISOString().split('T')[0];

    for (const doc of documents) {
      const fileBuffer = await this.minioService.downloadFile(doc.currentVersion.minioKey);
      const folderName = this.getDocumentTypeLabel(doc.type);
      const fileName = `${doc.title}_v${doc.currentVersion.versionNumber}.${this.getFileExtension(doc.mimeType)}`;

      zip.folder(folderName)?.file(fileName, fileBuffer);
    }

    // Agregar metadata.json con índice del legajo
    const metadata = {
      generatedAt: new Date().toISOString(),
      employeeId,
      documentCount: documents.length,
      documents: documents.map((d) => ({
        id: d.id,
        title: d.title,
        type: d.type,
        uploadedAt: d.uploadedAt,
        version: d.currentVersion.versionNumber,
      })),
    };

    zip.file('LEGAJO_INFO.json', JSON.stringify(metadata, null, 2));

    // Audit log
    await this.auditService.logAction({
      tenantId,
      documentId: 'LEGAJO_EXPORT',
      action: 'DOWNLOAD',
      performedBy: requestedBy,
      changeDetails: { type: 'legajo_zip', documentCount: documents.length },
    });

    return await zip.generateAsync({ type: 'nodebuffer', compression: 'DEFLATE' });
  }

      orderBy: { uploadedAt: 'desc' },

  tenantId: string;
  userId: string;
  role: AccessLevel;
  resourceType: 'DOCUMENT' | 'LEGAJO';
  resourceId: string; // documentId o employeeId
  action: 'UPLOAD' | 'DOWNLOAD' | 'DELETE' | 'VIEW' | 'VIEW_AUDIT';
}

@Injectable()
export class RoleBasedAccessControlService {
  constructor(private prisma: PrismaService) {}

  /**
   * Verificar si usuario tiene permisos para acción
   */
  async checkPermission(dto: PermissionCheckDto): Promise {
    const { tenantId, userId, role, resourceType, resourceId, action } = dto;

    // Obtener permisos del rol
    const rolePermissions = await this.prisma.documentRolePermission.findUnique({
      where: {
        tenantId_role: { tenantId, role },
      },
    });

    if (!rolePermissions) {
      throw new ForbiddenException('Role permissions not configured');
    }

    // Validar acción específica
    const hasPermission = this.validateAction(rolePermissions, action, resourceType, userId, resourceId);

    if (!hasPermission) {
      throw new ForbiddenException(
        `User role ${role} is not allowed to ${action} ${resourceType}`,
      );
    }

    return true;
  }

  /**
   * Lógica de validación por acción y rol
   */
  private validateAction(
    permissions: any,
    action: string,
    resourceType: string,
    userId: string,
    resourceId: string,
  ): boolean {
    switch (action) {
      case 'UPLOAD':
        return permissions.canUpload;

      case 'DOWNLOAD':
        return permissions.canDownload;

      case 'DELETE':
        return permissions.canDelete;

      case 'VIEW':
        if (permissions.canViewAll) return true;
        if (permissions.canViewOwn && this.isOwnResource(userId, resourceId)) {
          return true;
        }
        return false;

      case 'VIEW_AUDIT':
        return permissions.canViewAuditLog;

      default:
        return false;
    }
  }

  private isOwnResource(userId: string, resourceId: string): boolean {
    // Implementar lógica para verificar si es recurso propio

import { Injectable } from '@nestjs/common';
import { PrismaService } from '@/prisma/prisma.service';
import { DocumentType } from '@prisma/client';

interface LegajoView {
  employeeId: string;
  employeeName: string;
  documentsByType: {
    type: DocumentType;
    label: string;
    documents: any[];
    count: number;
  }[];
  totalDocuments: number;
  lastModified: Date;
  uploadProgress: {
    completed: number[];
    missing: DocumentType[];
  };
}

@Injectable()
export class LegajoViewService {
  // Tipos de documentos recomendados para completar legajo
  private readonly recommendedDocTypes = [
    DocumentType.CEDULA_IDENTIDAD,
    DocumentType.CONTRATO,
    DocumentType.CV,
    DocumentType.CERTIFICADO_ESTUDIO,
  ];

  constructor(private prisma: PrismaService) {}

  /**
   * Obtener vista completa de legajo por empleado
   */
  async getLegajoView(tenantId: string, employeeId: string): Promise {
    const employee = await this.prisma.employee.findUnique({
      where: { id: employeeId },
    });

    // Obtener documentos agrupados por tipo
    const documents = await this.prisma.document.findMany({
      where: {
        tenantId,
        employeeId,
        status: 'ACTIVE',
        deletedAt: null,
      },
      include: {
        currentVersion: true,
        metadata: true,
      },
      orderBy: { uploadedAt: 'desc' },
    });

    // Agrupar por tipo
    const documentsByType = this.groupDocumentsByType(documents);

    // Calcular estado de completado
    const completed = documentsByType.map((g) => g.type);
    const missing = this.recommendedDocTypes.filter((t) => !completed.includes(t));

    return {
      employeeId,
      employeeName: employee?.firstName + ' ' + employee?.lastName,
      documentsByType,
      totalDocuments: documents.length,
      lastModified: documents[0]?.uploadedAt || new Date(),
      uploadProgress: { completed, missing },
    };
  }

  private groupDocumentsByType(documents: any[]): any[] {
    const grouped = new Map();

    for (const doc of documents) {
      if (!grouped.has(doc.type)) {
        grouped.set(doc.type, {
          type: doc.type,
          label: this.getTypeLabel(doc.type),
          documents: [],
        });
      }
      grouped.get(doc.type).documents.push({
        id: doc.id,
        title: doc.title,
        uploadedAt: doc.uploadedAt,
        version: doc.currentVersion.versionNumber,
        size: Number(doc.fileSize),
      });
    }

    return Array.from(grouped.values()).map((g) => ({
      ...g,
      count: g.documents.length,
    }));
  }

  private getTypeLabel(type: DocumentType): string {
    const labels = {
      CONTRATO: 'Contratos Laborales',
      CV: 'Curriculum Vitae',
      CEDULA_IDENTIDAD: 'Cédula de Identidad',

      CONTRATO_BENEFICIOS: 'Contratos de Beneficios',
      DOCUMENTO_FISCAL: 'Documentos Fiscales',

    };

import { Injectable, Logger } from '@nestjs/common';
import { Cron } from '@nestjs/schedule';
import { PrismaService } from '@/prisma/prisma.service';
import { MinIOIntegrationService } from '../services/minio.integration';

@Injectable()
export class DocumentRetentionJob {
  private logger = new Logger(DocumentRetentionJob.name);

  constructor(
    private prisma: PrismaService,
    private minioService: MinIOIntegrationService,
  ) {}

  /**
   * Ejecutar cada día a las 2:00 AM
   * Eliminar documentos marcados para eliminación después de 90 días
   */
  @Cron('0 2 * * *')
  async handleDocumentRetention() {
    const retentionDays = 90;
    const retentionDate = new Date();
    retentionDate.setDate(retentionDate.getDate() - retentionDays);

    // Obtener documentos para eliminar
    const documentsToDelete = await this.prisma.document.findMany({
      where: {
        status: 'DELETED',
        deletedAt: { lte: retentionDate },
      },
      include: {
        versions: true,
      },
    });

    for (const doc of documentsToDelete) {
      try {
        // Eliminar versiones de MinIO
        for (const version of doc.versions) {
          await this.minioService.deleteFile(version.minioKey);
        }

        // Eliminar de BD
        await this.prisma.document.delete({
          where: { id: doc.id },
        });

        this.logger.log(`Hard-deleted document: ${doc.id}`);

        // Audit log
        await this.prisma.documentAuditLog.create({
          data: {
            tenantId: doc.tenantId,
            documentId: doc.id,
            action: 'DELETE',
            performedBy: 'SYSTEM',
            changeDetails: { type: 'hard_delete_retention' },
          },
        });
      } catch (error) {
        this.logger.error(`Failed to delete document ${doc.id}: ${error.message}`);
      }
    }
  }
}

8.2 Logging Centralizado

// src/modules/documents/services/document-audit.service.ts - Actualizado

import { Injectable } from '@nestjs/common';
import { PrismaService } from '@/prisma/prisma.service';
import { AuditService } from '@/audit/audit.service'; // Servicio central de auditoría

@Injectable()
export class DocumentAuditService {
  constructor(
    private prisma: PrismaService,
    private auditService: AuditService, // Inyectar servicio central
  ) {}

  async logAction(dto: {
    tenantId: string;
    documentId: string;
    action: string;
    performedBy: string;
    changeDetails?: any;
    ipAddress?: string;
    userAgent?: string;
  }): Promise {
    // Logging centralizado (para dashboard unificado)
    await this.auditService.log({
      tenantId: dto.tenantId,
      userId: dto.performedBy,
      resourceType: 'DOCUMENT',
      resourceId: dto.documentId,
      action: dto.action,
      details: dto.changeDetails,
      ipAddress: dto.ipAddress,
      userAgent: dto.userAgent,
    });

    // Logging específico de documentos (para compliance PDPA)
    await this.prisma.documentAuditLog.create({
      data: {
        tenantId: dto.tenantId,
        documentId: dto.documentId,
        action: dto.action,
        performedBy: dto.performedBy,
        changeDetails: dto.changeDetails,
        ipAddress: dto.ipAddress,
        userAgent: dto.userAgent,
        consentVerified: true, // Siempre registrar con consentimiento
      },
    });
  }
}
